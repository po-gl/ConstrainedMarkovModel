void MnemonicMarkovModel::applyConstraints ( vector< string >  constraint,
int  markovOrder 
)
virtual

Apply a constraint to the transition matrices by deleting nodes that violate the constraint.

The constraint is given as a string that represents the required first characters in order of chosen words

e.g. constraint="twd" constrains the model to only generate words starting with "T", "W", and "D" such as "The weather door"
